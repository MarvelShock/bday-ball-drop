What This Does

The overlay tracks midnight on July 16 as it rolls around the globe, starting at UTC+14 (Kiritimati / Line Islands) and ending at UTC−12 (Baker Island / AoE). Every time a timezone reaches July 16 midnight, the ball drops, fireworks, confetti, and sparkles fire, and "Happy Birthday Nicole" is displayed. Then the ball resets and waits for the next zone. This repeats automatically for all 31 representative worldwide time zones — a full 26-hour birthday celebration as the date circles the Earth. The four main U.S. time zones shown are New York (Eastern), Chicago (Central), Denver (Mountain), and Portland (Pacific).

The background is fully transparent for OBS chroma-key-free compositing. Only visible UI elements (cards, ball, text, particles) are colored.

OBS Setup Instructions

  1. In OBS, click + under Sources → choose Browser Source
  2. Paste the overlay URL from the Quick Copy box above
  3. Set Width: 900, Height: 600
  4. Enable "Refresh browser when scene becomes active"
  5. The background is transparent — no chroma key needed

Test Page Controls

Open overlay-test.html in a browser to access the test panel at the bottom:

  • 🎉 Full Celebration — triggers the full ball drop + fireworks/confetti/sparkles immediately
  • ⏭ Step Next Zone — advances one timezone and plays its celebration, then resets ball
  • Zone dropdown + ▶ Simulate Zone — jump to any specific timezone and simulate its midnight drop
  • ↺ Reset All — resets the entire sequence back to zone 1

A status indicator in the test bar shows which zone number you're currently on out of the total.
