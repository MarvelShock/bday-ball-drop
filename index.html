📺 OBS Setup Instructions

  1. In OBS, click + under Sources and select Browser Source.
  2. Paste the overlay URL: https://MarvelShock.github.io/bday-ball-drop/overlay.html
  3. Set Width to 900 and Height to 600.
  4. Check Refresh browser when scene becomes active, then click OK.
