# 🎂 Birthday Ball Drop Overlay

A **900×600 transparent stream overlay** that counts down to Nicole's birthday (July 16th) across every major timezone — and drops a glowing ball with fireworks, confetti, and a "Happy Birthday Nicole" celebration at midnight in each zone.

## Features
- 🌍 **16 global timezones** — all counting down simultaneously
- 🎱 **Ball drop animation** — glowing ball drops at midnight for each timezone
- 🎆 **Fireworks canvas** — multi-burst particle fireworks
- 🎊 **Confetti rain** — 180 colorful confetti pieces cascade down
- ✨ **Sparkle burst** — radial sparkle explosion on celebration
- 💛 **"Happy Birthday Nicole"** text overlay with gold glow
- 🫙 **Transparent background** — designed for OBS/Streamlabs as a browser source

## How to Use in OBS / Streamlabs

1. In OBS, add a **Browser Source**
2. Check **"Local file"** and point it to `overlay.html`, OR use the GitHub Pages URL
3. Set width: **900**, height: **600**
4. Enable **"Shutdown source when not visible"** and **"Refresh browser when scene becomes active"**

## GitHub Pages

Enable GitHub Pages in repo Settings → Pages → Deploy from `main` branch.  
Then use: `https://MarvelShock.github.io/bday-ball-drop/overlay.html`
